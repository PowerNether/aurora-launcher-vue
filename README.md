# Vurora Launcher

Разработка лаунчера на базе [Aurora Launcher](https://github.com/AuroraTeam/Launcher) с Electron Builder и Vue JS

## База проекта

Build tools

-   Vite
-   Electron Builder

Backend

-   Electron

Frontend

-   Vue
-   Vue Router
-   Pinia

UI Kit

-   Shadcn (Tailwind)

Code Style

-   Prettier

Организация кода

-   Atomic Design (Frontend)
